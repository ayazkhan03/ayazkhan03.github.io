About Project

Loan prediction is a very common real-life problem that each retail bank faces atleast once in its lifetime. If done correctly, it can save a lot of man hours at the end of a retail bank. I have performed Data cleaning, deal with outliers, EDA, preprocessing on data, Feature engineering and model building and achieve good accuracy and model is deployed using streamlit data science library.
