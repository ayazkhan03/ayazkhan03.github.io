About Project

Dataset contains 133 classes or breed of dogs. Performed data cleaning and data normalization. Split into train, validation and test. The next step I did training task and also used pretrain mode ResNet50 and achieved accuracy of 82.79 %. I have created streamlit web application and deployed at streamlit cloud.
