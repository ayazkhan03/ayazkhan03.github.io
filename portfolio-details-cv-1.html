About Project

Vehicle management is difficult task and monitoring each and every vehicle is impossible. To solve this problem, I have worked on project. It can manage vehicle entry and exit using number plate recognition. The model will detect number plate of the incoming and outgoing vehicle to read their number and check in record if not exist it make an entry of that vehicle. Additional to that it will also maintain and database of authorized vehicles and allow and restrict entry of vehicles accordingly. Technology Used: python 3, OCR, CNN, CV, Transfer Learning Using YOLO.
