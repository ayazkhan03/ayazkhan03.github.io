About Project

Now a days social media chatting is common among young people. Sentiment analysis is one of the important tasks to capture their sentiment while chatting. In this project I have analysis on various sentiment like happy, sad, neutral. I have done preprocessing like data cleaning, handling emoji, tokenization, stemming, converting short form into its original form (GN into good night and many more), TF-IDF, word2vec and then build a model and the can predict the new chat sentiment. I have achieved good accuracy.
